# chat
用rethinkdb搭建聊天

#### 1 安装rethinkdb [https://github.com/rethinkdb/rethinkdb]

#### 2 https://github.com/kenlyau/chat.git

#### 3 npm i

#### 5 npm run build

#### 4 npm start


## 用到的开源项目
- RethinkDB //nosql数据库
- Horizon //rethinkdb 开发的rethinkdb客户端
- Express //nodejs web 框架
- Vue // 前端单页应用组件化框架
